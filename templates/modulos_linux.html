Módulos para Ubuntu Linux

Abaixo estão alguns dos principais módulos Python para trabalhar com janelas, periféricos, e outros dispositivos no Linux:

  • tkinter - Para criar interfaces gráficas de utilizador (GUIs).
  • evdev - Para capturar eventos de dispositivos de entrada como teclados e ratos.
  • pyserial - Para comunicação via portas seriais.
  • pygame - Para captura de vídeo e som.
  • RPi.GPIO - Para controle de pinos GPIO em dispositivos como o Raspberry Pi.
